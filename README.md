## 프로젝트 배경
- 디지털정보화 수준이 가장 낮은 고령층을 타겟으로 설정하였다. 또한, 과학기술정보통신부의 2022년 디지털 정보격차 실태조사 결과에 따르면 고령층이 인터넷/모바일 서비스 경험이 없는 이유는 사용방법을 모르기 때문에 두번째로 높게 나타났다.
